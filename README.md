# Morse Kit

- [x] Play Morse Code using Flash, Sound and Vibration simultaneously. You can turn on/off any while playing. Implemented using Threads. 
- [x] Quick Table for Alpha Numeric charaters
- [x] Morse Code Encode-Decode
